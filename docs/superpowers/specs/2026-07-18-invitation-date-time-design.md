# 邀请函日期与到场时间调整设计

## 目标

将聚会日期统一调整为 2026 年 7 月 18 日（周六），并将大部队到场时间统一调整为 16:00–17:00。背景音乐保持现状。

## 修改范围

- `index.html`：更新页面标题、描述、Open Graph 文案、封面日期、三位普通来宾的到场时间、大部队到场时间和合影标题。
- `js/content.js`：更新活动日期、展示日期、大部队时间，以及三位普通来宾的到场时间。
- `js/app.js`：把分享文案中的“周日”更新为“周六”。
- `js/interactions.js`：把合影下载文件名中的“周日”更新为“周六”。
- `assets/og/party-preview.svg` 与 `assets/og/party-preview.jpg`：同步更新分享预览图中的星期文案，并保持现有尺寸与文件格式。
- `README.md`：把项目名称和功能说明中的“周日”更新为“周六”，避免文档与页面不一致。
- `tests/`：同步更新日期、星期、到场时间、分享文案和下载文件名相关断言。

## 保持不变

- `AI Native 连续创业者`仍为 `17:00 特别登场`。
- 主人组仍显示“已就位”。
- 地址、来宾身份、互动内容、视觉样式与背景音乐均不变。

## 验收标准

1. 页面公开文案不再出现旧日期 `2026.07.19` 或“周日”聚会描述。
2. 日期语义值为 `2026-07-18`，展示值为 `2026.07.18 周六`。
3. 大部队及三位普通来宾均显示 `16:00–17:00`。
4. 特别来宾仍显示 `17:00 特别登场`。
5. `npm test` 全部通过。
